Getting Started with phoneME Advanced MR2 Software

This guide contains information for building and initially testing a Java runtime environment based on Milestone Release 2 (MR2) of the phoneME Advanced project.

Most of the material provided here is for developers who wish to build, test or port phoneME Advanced technology for different kinds of devices, principally advanced phones. Application developers may wish to review the API reference documentation.
