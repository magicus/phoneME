 phoneME Feature Downloads
 
The phoneME Feature software includes the latest milestone and development implementations of CLDC and MIDP, as well as implementations of a number of optional package JSRs.  The latest code is available in the phoneME code repository.

phoneME Feature MR3

Below you will find the milestone release and supporting bundles for phoneME Feature MR3. MR3 includes implementations of CLDC 1.1.1, MIDP 2.1, and new optional package JSRs. Before you begin using phoneME Feature MR3, please read the Release Information and the Getting Started Guide.

 phoneME Advanced Downloads



The phoneME Advanced software includes the latest milestone and development implementations of CDC with related profiles and optional packages.  The latest code is available in the phoneME code repository.

phoneME Advanced MR1
phoneME Advanced MR1 software includes implementations of CDC 1.1.1, Foundation Profile 1.1, Personal Basis Profile 1.1, Personal Profile 1.1, plus an application management system.

Before you begin using MR1 software, please read the phoneME Advanced MR1 Release Information and Getting Started Guide.
